SenSiMet main goal is to create a software platform able to estimate the resemblance between two sentences and dynamically generate a graph based on the relationships between the terms in each sentence. With the use of PLN modules and semantic information platforms such as WordNet and BabelNet.

The main objectives of this project are the following:

  • Syntactically analyse sentences with PLN methods to identify the POS and type of each term
  • Automatically disambiguate each term meaning with Babelfy
  • Apply similarity metrics between the terms of each sentence
  • Automatically generate a graph based on the relationships between each sentence terms
  • Develop a REST API that allows anyone to use the developed methods and metrics
  • Develop a user-friendly interface with added information to further teach the user
